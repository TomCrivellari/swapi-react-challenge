# Desafio Frontend React

Aplicação desenvolvida como desafio técnico para vaga de estágio em Frontend, com foco em manipulação de dados, filtros dinâmicos, paginação e experiência do usuário.

-----------------------------------

## Funcionalidades

* Listagem de personagens via API (https://swapi.py4e.com/)
* Filtro por:
  * Nome
* Paginação de resultados
* Interface interativa e responsiva

-----------------------------------

## Tecnologias Utilizadas

* React
* TypeScript
* JavaScript (ES6+)
* CSS
* Fetch API

-----------------------------------

## Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/TomCrivellari/swapi-react-challenge.git

# Acesse a pasta
cd swapi-challenge

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```

-----------------------------------

## Objetivo

Demonstrar domínio em:

* Construa uma interface que liste os personagens;
* A tela, além da grid, deverá ter um filtro que permita filtrar pelo nome dos mesmos;
* O grid deverá ser paginado, exibindo 10 registros por vez, e botões de navegação entre os mesmos;
